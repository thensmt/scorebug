Operator Guide

Welcome to NSMT Stats
This guide walks you through every screen in the app. If it's your first time, read this before your first game. It takes about 5 minutes.
1
Set Up Your Roster (Do This Before Game Day)
Go to Roster Manager from the home screen. Tap + New to create a roster for each team you cover regularly. Enter the team name, short code (e.g. PVI), and every player's jersey number and full name. Tap Save Roster. Rosters are saved permanently and can be reused every game — you'll never enter them again.
💡 Tip: Save both teams' rosters before the season starts. During games you'll load them in seconds.
2
Start a New Game
Tap New Game → enter your name → select your mode:

Stat-Keeper Mode — tracks both teams. Use this for official NSMT coverage.
Coach Mode — tracks one team only. Use this when working with a coaching staff.
💡 Tap Continue, then fill in the event name, venue, date, and time on the Game Setup screen.
3
Load Your Rosters
On the Game Setup screen, tap 📋 Load Roster next to each team. Pick the saved roster from the list — all players populate instantly. If a team has a new player not in your saved roster, tap + Add Player to add them manually. When done, tap 💾 Save Rosters to update them for future games, then tap ▶ Start Game.
4
The Live Game Screen
The live screen has three parts:

Scoreboard (top) — shows live score, period, and team foul dots.
Three tabs — Log Event (main operator view), Box Score (live stats table), Play-by-Play (event log).
Bottom bar — Undo, Clock, Period, Report, End Game.
5

Every stat is logged in steps. The flow bar at the top shows where you are.

Step 1: Tap the player's jersey number (left column = away, right = home).
Step 2: Tap the action — 2PT, 3PT, Free Throw, Rebound, Steal, etc.
Step 3: For shots/FTs only — tap MADE ✅ or MISSED ❌.
Step 4: For made field goals — tap the shot location (Paint / Mid / 3PT) and play context (Half Court / Fast Break / 2nd Chance / Off Turnover). Default is Half Court — just tap Next if no context.
Step 5: Tap the assisting player if there was one, or tap No Assist. Stats log instantly.
💡 For non-shot stats (Rebound, Steal, Block, TO, Foul) — it logs immediately after Step 2. No extra taps needed.
6
Undo a Mistake
Tap ↩ Undo in the bottom bar at any time to remove the last logged event. The stats and score update immediately. You can only undo one event at a time — so correct mistakes as soon as you notice them.
💡 If you're mid-flow and selected the wrong player, tap ← Back in the action panel to go back without logging anything.
7
Managing Periods
At the end of each quarter or half, tap ⏱ Period in the bottom bar. Enter the official period scores if you have them (optional — useful for report accuracy), then tap Next Period →. Team fouls reset automatically. To add overtime, tap + Add Overtime instead.
8
Setting the Clock
Tap 🕐 Clock and enter the current game time in MM:SS format (e.g. 6:32). This timestamps your play-by-play entries and helps with game flow in the report. You don't need to track it every possession — just update it at key moments like timeouts or end of period.
9
Ending the Game & Generating the Report
Tap End in the bottom bar → confirm the final score → tap End Game & Generate Report. The official box score report generates automatically with full stats, advanced summary blocks, and period scores. Tap 🖨️ Print / Save PDF to export it. The game saves to your archive automatically.
💡 You can also tap 📋 Report at any point during the game to preview stats without ending it.
10
Exporting & Saving
From the Report screen, tap 📦 Export .nsmtgame to download a full game file. This is your backup and can be shared with NSMT HQ. All games also auto-save in your Game Archive on this device so you can reopen them anytime.

🔴 Emergency Procedures
App crashes mid-game: Re-open the app. Tap Game Archive or the recent game on the home screen. All events are auto-saved — you won't lose stats.

Wrong player logged: Tap ↩ Undo immediately. Re-log with the correct player.

Lost Wi-Fi: No impact. The app works fully offline. All data is stored on the device.

Need to correct a past event: You cannot edit individual past events in v1. Use Undo as you go. If a systematic error happened, contact NSMT HQ with the exported game file.

Stats Glossary
Every stat tracked by NSMT Stats and how it's defined.
Basic Box Score
PTS
Points — Calculated automatically: (2PT Made × 2) + (3PT Made × 3) + FT Made
FGM / FGA
Field Goals Made / Attempted — All 2PT and 3PT attempts combined
3PM / 3PA
3-Pointers Made / Attempted — Shots taken from behind the arc
FTM / FTA
Free Throws Made / Attempted — Log each attempt individually
OFF / DEF
Offensive / Defensive Rebounds — Off Reb = team keeps possession; Def Reb = team gains possession
REB
Total Rebounds — Offensive + Defensive combined
AST
Assists — A pass directly leading to a made basket. Tagged on the made shot in Step 5
STL
Steals — Logged to the player who takes the ball from the opponent
BLK
Blocks — Logged to the player who deflects the shot
TO
Turnovers — Logged to the player who loses the ball (bad pass, travel, charge, etc.)
PF
Personal Fouls — Logged to the player who commits the foul. Also increments team foul counter
Advanced / Summary Stats
Paint Pts
Points in the Paint — Made 2PT shots tagged with location = Paint
Fast Break
Fast Break Points — Made baskets tagged with context = Fast Break
2nd Chance
Second Chance Points — Made baskets tagged with context = 2nd Chance (after offensive rebound)
Off TO
Points Off Turnovers — Made baskets tagged with context = Off Turnover
FG%
Field Goal Percentage — FGM ÷ FGA × 100. Auto-calculated in reports
3PT%
3-Point Percentage — 3PM ÷ 3PA × 100. Auto-calculated
FT%
Free Throw Percentage — FTM ÷ FTA × 100. Auto-calculated
App-Specific Terms
.nsmtgame
Game Export File — The full game data file. Send this to NSMT HQ after each game for the central archive
Game Pack
Pre-loaded Game File — (Future feature) Admin-created file with rosters and game info pre-filled for operators
Flow Bar
Step Indicator — The bar at the top of the Log screen showing which step of event logging you're on
Stat-Keeper Mode
Full Game Mode — Tracks both teams. Used for official NSMT coverage and report generation
Coach Mode
Single Team Mode — Tracks one team only. Used when working directly with a coaching staff
